# 商业落地与产品化规划

## 1. 产品定位
- **定位**：面向科研/教学的“论文级光路图与实验记录系统”。
- **核心价值**：仿真 + 专业绘图 + 论文模板 + 高质量导出，一体化完成投稿级图形。

## 2. 目标客户
- 高校与科研机构（研究生、博士后、课题组）
- 企业研发部门（激光/光学/光电子）
- 本科教学实验课程

## 3. 产品形态
- **Web 版**：低门槛试用、快速传播。
- **桌面版**：本地存储、离线使用、稳定授权与批量部署。

## 4. 付费与授权（建议）
- 学生版（个人）：年付
- 个人专业版：年付
- 课题组授权：人数梯度
- 机构版：批量部署/离线授权

## 5. 商业落地路径
1) **MVP 商业化**
   - 统一版本号、变更记录、许可说明
   - 5-10 套论文级模板 + 3 个演示场景
   - 落地页 + 演示视频 + 价格页
2) **桌面版发布**
   - Electron 打包（离线、稳定、可控授权）
   - 本地项目库 + 导出管理
3) **渠道与分发**
   - 官网下载、学术社区推广、导师/课题组试用
   - 教学场景合作（实验课示例与模板）

## 6. 运营与增长
- **种子用户**：目标 5-10 个课题组试用
- **反馈闭环**：导出质量、模板适配、批量修改效率
- **增长资料**：论文级图例库、案例拆解、演示视频

## 7. 研发里程碑
- **M0（当前）**：功能完整性与一致性验证
- **M1（2-4 周）**：桌面版打包 + 本地项目库稳定
- **M2（4-8 周）**：模板库扩充 + 文档/演示完善
- **M3（8-12 周）**：授权与订阅、团队版支持

## 8. 合规与风险
- 素材与图标许可清单
- 论文模板尺寸与字体规范校验
- 离线授权与许可证策略
- 数据本地化与隐私合规

## 9. 落地交付清单
- 桌面版安装包（Win）
- 产品手册（用户指南 + 快速上手）
- 论文级模板库与示例图集
- 价格页与商业落地页
